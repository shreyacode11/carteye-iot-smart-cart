# Smart Cart 🛒

A smart shopping cart system integrating **Arduino sensors**, **Firebase**, and a **web app** to monitor items in real-time.

## Features
- Real-time weight sensing using **HX711 load cell**
- Distance detection using **ultrasonic sensor**
- Data synced to **Firebase Realtime Database**
- Web app interface built with **React / Vite**
- Option to integrate AI APIs for smarter interactions

## Tech Stack
- **Hardware:** ESP32, HX711, Ultrasonic Sensor
- **Software:** React / Vite / Firebase / Node.js
- **Programming:** JavaScript, C++, HTML, CSS
